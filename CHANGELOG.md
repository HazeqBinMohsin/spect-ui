## [1.3.3] — 2026-MAY-14 — LATEST

### Fixed

- Fixed the white text color bug in *Spect (Light Classic)*

- Fixed "too dull colors" in *Spect (Dark Classic)*

### Released

- Released *Spect (Dark Colorblind)* color theme

### Improved

- Icon theme *Spect (Light)* has more saturated and darker colors for visibility on light backgrounds

## [1.3.2] — 2026-MAY-13

### Added

- Added seperate icons for Py and Pym (PY Module) files

### Released

- Released *Spect (Dark Professional)* color theme

- Released *Spect (Light Classic)* color theme

- Released *Spect (Light)* icon theme

### Improved

- Made basic changes to existing color themes (brightened the text up, and made them more vibrant)

## [1.3.1] - 2026-MAY-07 — LATEST

- Added GitHub repo and issue templates
- Created tagline **(Built By Professionals, For Professionals)**
- Updated Extension Aesthetics
- Updated Spect branding

### Improved 

- Improved VSCode version compatibility to support older versions

## [1.2.9] - 2026-MAY-06 — LATEST

### Added

- Added icons for Bun files, Minecraft worlds, Bezier Curves, and Applescript files

### Changed

- Improved icon for README files
- Replaced `.vscode` folder icon with the VSCode logo
- Replaced Audio *Play* icon with *Music Note*

## [1.2.8] - 2026-MAY-02

### Changed

- Renamed *Spect (Dark)* to *Spect (Dark Classic)*

- Replaced ALL previous icons with slightly larger ones

### Added

- Added icons for `.RSS`, `.WASM`, `.VSIX`, and `.CSPROJ` files

- Added icons for VSCode, Symlink and Cursor files
